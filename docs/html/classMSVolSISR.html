Detailed Description

A SISR implementation of the multivariate stochastic volatility model of Pitt and Shephard 1999.

Author
taylor
Date
08/09/17

Parameters
numPartsan int describing the number of particles.
betaa Mat representing the factor loadings matrix.
phisa Vec representing the AR parameters of the log-volatility processes.
musa Vec representing the mean of the log-volatility processes.
sigmasa Vec representing the standard deviation of the log-volatility processes.
resampTechniquean enum class that determines the type of resampling to be performed.
pathLengthan int. Set to 0 if jus filtering. To store entire path samples, set to time-length of data.
percEssthe percentage effective sample size resampling threshold.
