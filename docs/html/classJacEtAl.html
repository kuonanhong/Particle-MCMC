Detailed Description

A SISR implementation of the multivariate stochastic volatility model of Jacquier et al.

Author
taylor
Date

Parameters
numParts,aninteger representing the number of particles.
beta,aMat representing the factor loadings matrix.
RAsVec,aVec representing the diagonal elements of the observational covariance matrix.
mus,aVec representing the mean of the log-volatility processes.
phis,aVec representing the AR parameters of the log-volatility processes.
sigmas,aVec representing the standard deviations of the log-volatility processes.
resampTechnique,anenum class representing the type of resampling to be performed.
pathLength,setto the length of your time series data if you want to store entire paths. Otherwise set to 0.
percEss,thepercent of the maximum effective sample size threshold for resampling. Not important for several resampTechniques.
